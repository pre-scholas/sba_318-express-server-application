# SBA_318-express-server-application

Personal Blogging Platform API:

1. Concept: Create a backend API to manage blog posts, including operations like creating, retrieving, updating, and deleting articles (CRUD operations).
2: Skills Developed: Understanding of resources and endpoints, HTTP methods (GET, POST, PUT, DELETE), database interaction (SQL or NoSQL), and basic server-side logic.

## Objective

. Create a server application with Node and Express.
. Create a RESTful API using Express.
. Create Express middleware.
. Use Express middleware.
. Use a template engine to render views with Express.
. Interact with a self-made API through HTML forms.
